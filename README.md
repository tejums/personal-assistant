# Daemon

Daemon is a personal AI executive assistant that attends stand-up meetings, processes Recall.ai transcripts, detects when the user is mentioned, and generates summaries and action items.

## Features

- Attends stand-up meetings automatically
- Processes meeting transcripts from Recall.ai
- Detects mentions of the user in conversations
- Produces concise summaries of key discussion points
- Extracts actionable follow-ups and tasks

## How It Works

1. Daemon joins or ingests stand-up meeting sessions.
2. It retrieves and processes Recall.ai transcript data.
3. It identifies moments where the user is explicitly mentioned.
4. It generates a meeting summary.
5. It compiles action items based on discussion context.

## Use Cases

- Stay up to date on stand-up outcomes when multitasking
- Track decisions and commitments from daily syncs
- Avoid missing tasks assigned in conversation

## Project Status

Early-stage project. Core functionality focuses on transcript processing, mention detection, and summary/action-item generation.
